# 项目简介

## 核心功能
- 用户登录注册
- 学生管理
- 名师管理
- 课程名称管理
- 课程信息管理
- 学院管理（如计算机学院、设计学院等）

## 适用场景
- 在线教育机构
- 教育培训公司
- 个人教师或教育机构

## 技术特点
- 采用Python编程语言
- 使用Django框架
- 数据库操作通过SQLAlchemy进行管理
